# Changelog

本專案版本號採 [SemVer](https://semver.org/lang/zh-TW/)：`vMAJOR.MINOR.PATCH`。
每次發版請在最上方新增一個區段，bot 會把「目前 `app/version.py` 版本」對應的區段
自動推播給已開通使用者（見 `app/version.py` 的改版流程）。

## [1.1.0] - 2026-06-29
- 新增 /version 指令：可隨時查看目前版本與更新內容。
- 啟用「改版自動通知」：日後有新版本會主動推播更新內容給你 🎉

## [1.0.0] - 2026-06-29
首個正式版本。
- Telegram 貼網址追蹤購物網商品價格，變動時提醒（只通知該商品擁有者）。
- 分層萃取管線（專屬 adapter → 結構化資料 → Playwright 渲染），執行時不串 LLM。
- 互動式指令 + 輸入框選單；多步驟對話、30 秒逾時自動取消。
- 白名單授權：`/start` 靜默記錄、管理員 `/allow` 動態開通。
- `/status` 顯示價格走勢圖、最高/最低與漲跌紀錄。
- 清單顯示購物網名稱與最近爬文時間。
- 重複追蹤偵測、爬蟲失效提醒，管理員待辦清單與 `/add-scraper` 快速新增爬蟲。
